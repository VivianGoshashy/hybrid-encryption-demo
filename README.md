# Hybrid Encryption Demo

A practical demonstration of hybrid encryption combining AES and RSA.

## Features
- RSA key generation (2048-bit)
- AES file encryption (Fernet)
- RSA key encryption for secure exchange
- SHA-256 integrity validation

## Installation
```bash
pip install cryptography tabulate
```

## Usage
```bash
python3 encryption_demo.py
```
